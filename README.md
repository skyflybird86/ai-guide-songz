# Dream AI 企业智能体平台

## 项目简介

Dream AI 企业智能体平台是面向企业客户的智能化解决方案，通过先进的AI技术帮助企业实现业务流程自动化、客户服务智能化和数据分析精准化。

## 技术特点

- 构建企业大脑和数字员工
- 提升企业智能化水平
- 基于大模型算法与企业专有知识深度融合
- 提供智能助手、智能问答及多种智能应用服务

## 功能模块

- 企业AI智能体平台应用
- 定制智能体应用
- 数字化项目服务
- 客户案例展示

## 部署方式

### 本地运行

```bash
cd company_website
python3 -m http.server 8000
```

访问 http://localhost:8000 查看网站

### 线上部署

使用 Vercel 一键部署：

## 文件结构

- `index.html`: 主页面
- `css/final-perfect-style.css`: 样式文件
